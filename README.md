# personal-website

Personal site for rafaellopezgarcia.com, built with React + Vite.

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment (Vercel)

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Domain: `rafaellopezgarcia.com` (managed via Cloudflare)
